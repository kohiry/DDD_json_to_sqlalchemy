# blog

The simple proj.

## Stack

- **framework**: FastAPI
- **migrations**: Alembic
- **ORM**: Sqlalchemy
- **Dependencies manager**: Poetry
- **Automation**: Docker, Makefile
- **Architecture**: similar to DDD

### How to set up?
1) Change .env.example to .env (look to env dir)
2) `make up` (see more command in Makefile)
